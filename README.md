# Đập Chuột ⭐

Game đơn giản: click để đập chuột bằng búa, sao bay lung tung khắp màn hình!

## Chơi thử local

Mở file `index.html` trên trình duyệt, hoặc chạy:

```bash
python3 -m http.server 8080
```

Rồi vào http://localhost:8080

## Host lên GitHub Pages

1. Push code lên GitHub
2. Vào **Settings** → **Pages**
3. Chọn **Source**: Deploy from a branch
4. Chọn branch **main** (hoặc **master**), folder **/ (root)**
5. Bấm **Save** — vài phút sau site sẽ có tại `https://<username>.github.io/<repo-name>/`
